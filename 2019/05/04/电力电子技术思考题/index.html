第二章 电力电子器件

概述

电力电子器件有哪些特征
       1）电力电子器件所能处理的电功率的范围（所承受电压电流的范围）很广，一般小至毫瓦级，大至兆瓦级。一般都远大于处理信息的电子器件
       2）一般都工作在开关状态。因为处理的电功率比较大，为了减小本身的损耗，提高效率。
       3）在实际工作中，电力电子器件往往需要由信息电子电路来进行控制，并且信息电子电路的需要还应当通过放大电路进行放大（驱动电路）。
       4）电力电子器件的自身损耗一般远大于信息电子器件，在工作时一般还需要安装散热器。

电力电子器件的损耗分为哪几种，占主要的是什么类型的损耗
       四种，分别为通态损耗、断态损耗、开通损耗、关断损耗，后两者又统称开关损耗。一般来说，通态损耗是电力电子器件损耗的主要原因；但当器件的开关频率较高时，开关损耗也会随之增大而可能成为器件功率损耗的主要原因。

电力电子系统中主电路和控制电路之间为什么需要有电气隔离，有哪些电气隔离的方法
       电力电子在实际应用中，一般是由控制电路、驱动电路和以电力电子器件为核心的主电路组成一个系统。主电路的电压和电流一般都大，而控制电路的元器件只能承受较小的电压和电流，因此在主电路和控制电路连接的路径上，如驱动电路和主电路的连接处，或者驱动电路与控制信号的连接处，以及主电路与检测电路的连接处，一般需要进行电气隔离，而通过其他手段如光、磁来传递信号。

电力电子系统中驱动电路和保护电路的作用是什么
       驱动电路：放大控制电路的控制信号，以控制主电路中电力电子器件的导通或关断，来完成整个系统的功能；保护电路：主电路中往往存在电压和电流的过冲，而电力电子器件承受过电压和过电流的能力较弱，因此在主电路和控制电路中附加一些保护电路，以保证整个电力电子系统的正常可靠运行是十分必要的。

按被控制信号的控制程度，可将电力电子器件分为哪几种类型
       半控型器件、全控型器件、不可控器件。半控型：能控制开通，不能控制关断；全控型：能控制开通，也能控制关断；不可控型：开通和关断均不能控制。

电平触发型和脉冲触发型器件有什么区别
       二者的驱动电路加在电力电子器件控制端和公共端之间有效信号的波形不同。施加一个电压或者电流的脉冲信号来实现器件的开通或者关断的为脉冲型触发型器件；施加一定电平的电压或电流信号来使开通并维持在导通状态，或者关断并维持在阻断状态的，为电平控制型器件

单极型器件和双极型器件的区别是什么
       其内部电子和空穴两种载流子参与导电的情况不同。单极型：由一种载流子参与导电；双极型：由电子和空穴两种载流子参与导电；复合型：单极型和双极型复合而成的器件

电力二极管（不可控器件）

PN结中空间电荷区是怎样形成的
       由于N区和P区交界处电子和空穴浓度的差别，造成了各区的多数载流子（多子）向另一区移动的扩散运动，到对方区内称为少数载流子（少子），从而在界面两侧分别留下了不能任意移动的杂质离子（被称为空间电荷）。空间电荷建立的电场被称为自建电场或内电场，其方向是阻止多子的扩散运动，另一方面又吸引对方区的少子向本区移动（少子的漂移运动）。扩散运动和漂移运动最终达到动态平衡，正负空间电荷量也达到稳定值，形成了一个稳定的由空间电荷构成的范围，被称为空间电荷区，按所强调的角度不同也被称为耗尽层、阻挡层或势垒区。

分析PN结中的单向导电原理
       当PN结外加正向电压时（正向偏置），即外加电压的正端接P区，负端接N区，外加电压场与PN结自建电场相反，使得多子的扩散运动大于少子的漂移运动，形成扩散电流，在内部造成空间电荷区变窄，而在外电路上形成自P区流入而从N区流出的电流，称为正向电流$I_F$。当外加电压升高时，自建电场将进一步被削弱，扩散电流进一步增加。这是PN结的正向导通状态。
       当PN结外加反向电压时（反向偏置），即外加电压的正端接N区，负端接P区，外加电压场与PN结自建电场相同，使得少子的漂移运动大于多子的扩散运动，形成漂移电流，在内部造成空间电荷区变宽，而在外电路上则形成自N区流入而从P区流出的电流，成为反向电流$I_R$。但少子的浓度很小，在温度一定时漂移电流的数值趋于恒定，被成为反向饱和电流$I_S$，一般仅为微安数量级，因此反向偏置的PN结表现为高阻态，几乎没有电流通过，被称为反向截止状态。

电力二极管的结构中的低参杂N区起什么作用
       低参杂N区（在半导体物理中用$N^-$表示），也称漂移区。由于参杂浓度低，低参杂N区就可以承受很高的电压而不致被击穿，因此低参杂N区越厚，电力二极管能承受的反向击穿电压就越高。

什么是电导调制效应
       当PN结上流过的正向电流较小时，二极管的电阻主要是作为基片的低参杂N区的欧姆电阻，其阻值较高且为常量，因而管压降随正向电流的上升而增加；当PN结上流过的正向电流较大时，由P区注入并积累在低参杂N区的少子空穴浓度也相应大幅增加，使得其电阻率明显下降，即电导率大大增加，这就是电导调制效应。

电力二极管正向导通时，即使正向电流有较明显的变化，正向电压降也只有微小的变化。试分析原因。
       电导调制效应：当PN结上流过较大的正向电流时，由P区注入并积累在低参杂区的少子空穴浓度将很大，为了维持半导体的电中性条件，其多子的浓度也将大幅度增加，使得其电阻率明显下降，即电导率大大增加。

在电力二极管由正向偏置转换为反向偏置时，会产生较大的反向恢复电流，为什么？
       当由正向偏置转为反向偏置时，电力二极管由于在PN结两侧（特别是多参杂N区）储存有大量少子的缘故而并没有恢复反向阻断能力，这些少子在外加反向电场的作用下被抽取出电力二极管，因而流过较大的反向电流。

恢复系数的定义
       电力二极管的动态特性中，下降时间和延迟时间的比值$t_f/t_d$被称为恢复特性的软度或者恢复系数，应用$S_r$表示。

电力二极管的主要类型有哪些
       1）普通二极管（整流二极管）；2）快速恢复二极管；3）肖特基二极管

电力二极管选型时，应怎样选择它的反向重复峰值电压$U_{RRM}$
       该电压是指对电力二极管所能重复施加的反向电压最高峰电压。通常是其雪崩击穿电压$U_B$的$2/3$。使用时，往往按照电路电力二极管可能承受的反向最高峰值电压的两倍来选定此参数。

晶闸管的关断条件是什么

什么是晶闸管的正向恢复时间/正向阻断恢复时间
       晶闸管的关断过程所需的关断时间$tq$可以分为反向恢复时间$t{rr}$和正向恢复时间$t_{gr}$

  • $t_{rr}$：从正向电流降为0，到反向恢复电流衰减至接近于0的时间。
  • $t_{gr}$：反向恢复过程结束后，由于载流子复合过程较慢，晶闸管要恢复其对正向电压阻断能力所需要的时间。在该时间内，如果重新对晶闸管施加正向电压，晶闸管会重新导通，而不是受门极触发电流控制而导通。

GTO和传统晶闸管相比，器件结构上主要由什么不同

GTO的主要缺点是什么

电力晶体管/GTR（全控型器件）

怎样判断GTR进入了饱和状态
       集电极电压$u_c$小于基极电压$u_b$

GTR的主要缺点是什么
       驱动电路损耗大，易于二次击穿，可靠性低。

电力场效应晶体管/电力MOSFET（全控型器件）

为什么电力MOSFET的导通电阻随器件耐压升高迅速增大
       栅极和P区间是绝缘的，因此无电导调制效应。因此电力MOSFET的耐压低，一般在1000V以下，多用于功率不超过10kW的电力电子装置。

为什么电力MOSFET关断过程非常迅速
       电力MOSFET只靠多子导电，不存在少子储存效应，因而其关断过程是非常迅速的。

电力MOSFET是场控器件，但驱动电路仍然需要较大的电流输出能力，为什么
       在开关过程中，需要对输入电容充放电，仍需要一定的功率，且开关频率越高，所需要的驱动功率越大。

绝缘栅双极晶体管/IGBT（全控型器件）

IGBT关断时为什么会有拖尾电流
       关断时，储存的少子复合，形成拖尾电流

IGBT是单极型器件还是双极型器件
       是由GRT（双极性）和电力MOSFET（单极型）连接而成，因此为混合极型器件。

电容滤波不可控整流电路

电容滤波的单相不可控整流电路，交流输入电压有效值为$U_2$，空载时输出电压为多少？随负载加重，输出电压逐渐趋近于多少？
       空载时，输出电压最大，为$U_d=\sqrt2 U_2$，随着负载加重，电容放电加快，几乎失去储能作用，电压逐渐趋近于$0.9U_2$。

       空载时，输出电压最大，为$U_d=\sqrt6 U_2$，即$2.45U_2$；随着负载加重，输出电压平均值逐渐趋近于$2.34U_2$。

无功功率会对公用电网带来哪些不利影响
1）无功功率会导致电流增大和视在功率增加，从而使设备和线路的损耗增加；
2）无功功率增加，会使总电流增加，从而使设备和线路的损耗增加；
3）无功功率使线路压降增大，冲击性无功负载还会使电压剧烈波动。

谐波会对公用电网造成什么危害
1）谐波使电网中的元件产生附加的谐波损耗，降低发电、输电及用电设备的效率，大量的三次谐波流过中性线会使线路过热甚至引发火灾；
2）谐波影响各种电气设备的正常工作，使电机发生机械振动、噪声过热，使变压器局部过热，使电容器、电缆等设备过热、使绝缘老化、寿命缩短以致损坏；
3）谐波会引起电网中局部的并联谐振和串联谐振，从而使谐波放大，会使上述1）和2）所述危害放大，甚至引起严重事故；
4）谐波会导致继电保护和自动保护装置的误动作，并使电气测量仪表计量不准请；
5）谐波会对邻近的通信设备产生干扰，轻者产生噪声，降低通信质量，重者导致信息丢失，使通信系统无法正常工作。

换流方式

和晶闸管相关的换流方式有哪几种？
       换流方式一共有四种：器件换流、电网换流、负载换流、强迫换流（电容换流）。在这四种换流方式中，器件换流只针对全控型器件，其余三种换流方式主要是针对晶闸管而言的。

电压型和电流型逆变电路的主要特点分别是什么？
电压型逆变电路的特点：
1）直流侧为电压源，或并联大电容，相当于电压源。直流侧电压基本无脉动，直流回路呈现低阻抗。
2）由于直流电压源的钳位作用，交流侧输出电压波形为矩形波，并且与负载阻抗角无关。而交流侧输出电流波形和相位因负载阻抗情况的不同而不同。
3）当交流侧为阻感负载时需要提供无功功率，直流侧电容起缓冲无功能量的作用。为了给交流侧向直流侧反馈的无功能量提供通道，逆变桥各臂都并联了反馈二极管。
电流型逆变电路的特点：
1）直流侧为电流源，或串联大电感，相当于电流源。直流侧电流基本无脉动，直流回路呈现高阻抗。
2）电路中开关器件的作用仅是改变直流电流的流通路径，因此交流侧输出电流为矩形波，并且与负载阻抗角无关。而交流侧输出电压波形和相位因负载阻抗情况的不同而不同。
3）当交流侧为阻感负载时需要提供无功功率，直流侧电感起缓冲无功能量的作用。因为反馈无功能量时直流电流并不反向，因此不必像电压型逆变电路那样要给开关器件反并联二极管。

有哪六种基本斩波电路

       一共三种控制方式：脉冲宽度调制、频率调制或者调频型、混合型

什么是电感伏秒平衡原理
       当开关电路稳定工作的时候，一个开关周期内的电感的电流变化量最终为0，即经过电感的电流增量和电感的电流减少量是相等。换句话说，在开关电路中，一个周期因开关作用被分为两段，其中一段时间内电感电流在增加，另一段时间内电感电流在减少，稳定状态下那么在一个开关周期内，电流的增加量与电流的减少量是相等。

同步整流的好处是什么
       同步整流可提高效率，同时也能够极大地帮助瞬态负载调节。它为电源预加载提供了一种高效的方法。另外，相比摆动电感，它还拥有更加稳定的控制环路特性。它提高了传统降压转换器，以及所有其他能够使用同步整流的拓扑结构的动态性。

分析将开关电源工作频率提高可以减小其体积和重量的原因
       因为感抗公式 $X_L = \omega L = 2\pi fL$ 要得到相同的感抗，更高的频率可以使用更小的电感，也就是电感的圈数越少即体积和重量就小了。

第六章 交流-交流变流电路

交流电力控制电路分为哪几种
       交流电力控制电路指只改变电压、电流或对电路的通断进行控制，而不改变频率的电路。交流电路控制分为：交流调压电路、交流调功电路和交流无触点开关。

单相交-交变频电路在一个输出电压周期中分为哪几个时间段？
一共可以分为6个时段，见教材P151：
1）$i_0<0$、$u_0>0$：反组逆变
2)电流过0，为切换死区
3）$i_0>0$、$u_0>0$：正组整流
4）$i_0>0$、$u_0<0$：正组逆变
5）电流过0，切换死区
6）$i_0<0$、$u_0<0$：反组整流

单相交-交变频电路的输出上限频率约为多少？输出频率过高时会有什么问题
       一般认为，输出上限频率不高于电网频率的1/3~1/2。电网频率为50Hz时，交-交变频电路的输出上限约为20Hz。输出频率增高时，输出电压一周期所含电网电压的段数就减少，波形畸变就更严重。

带阻感负载的单相交流调压电路中，为什么需要触发脉冲有足够的宽度
       $\alpha$的移相范围为$\varphi$~180°，必须防止由于控制角小于阻抗角造成的输出交流电压中出现直流分量的情况。如果触发脉冲过窄，当$\alpha<\varphi$时会发生有一个晶闸管无法导通的现象，电流出现很大的直流分量。

PWM控制技术

解释：面积等效原理；异步调制；同步调制；特定谐波消去法；直流电压利用率
       如下

  • 面积等效原理：冲量相等而形状不同的窄脉冲加在具有惯性的环节上时，其效果基本相同；冲量即窄脉冲的面积，所说的效果基本相同是指环节的输出波形基本相同。如果把各输入波形用傅里叶变换分析，则其低频段非常接近，仅在高频略有差异。面积等效原理是PWM控制技术的重要基础理论。
  • 异步调制：载波信号和调制信号不保持同步的调制方式。该方式希望采用较高的载波频率，以使在信号波频率较高时仍能保持较大的载波比。
  • 同步调制：载波比$N$为常数，并在变频时使载波和信号波保持同步的方式
  • 特定谐波消去法：首先尽量使波形具有对称性，为消去偶次谐波，应使波形正负两个半周期对称，为消去谐波中的余弦项，使波形在正半周期前后1/4周期以1/2为轴线对称
  • 直流电压利用率：直流电压利用率是指逆变电路所能输出的交流电压基波最大幅值$U_{1m}$和直流电压$U_d$之比。提高直流电压利用率可以提高逆变电器的输出能力。

采用自然采样法时，单极性SPWM和双极性SPWM的区别是什么
       单极性：三角波载波在信号波正半周期或负半周期里只有单一的极型，所得的SPWM波在半个周期内也只在单极型范围内变化；电压输出有$\pm U_d$和$0$。
       双极性：三角波载波始终是有正有负为双极性的，所得的SPWM波形在半个周期内有正有负；电压输出仅有$\pm U_d$
